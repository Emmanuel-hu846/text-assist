# Message Auto-Reply System

## Quick Start

### 1. Build and Start
```bash
docker-compose up --build
```

### 2. Prepare Your Message Data
Export your WhatsApp chats:
- WhatsApp → Contact → More → Export chat (without media)
- Save as `chat_export.txt`

### 3. Train the Model
```bash
docker-compose exec auto-reply python3 train_model.py /app/messages/chat_export.txt /app/models/training_messages.json
```

### 4. Run the Engine
```bash
docker-compose exec auto-reply ./message_engine
```

## Files

- `message_engine.cpp` - Core Markov chain text generator
- `train_model.py` - Parse WhatsApp/JSON messages into training data
- `integration_daemon.py` - Monitor messages and auto-reply with confidence scoring
- `docker-compose.yml` - Container orchestration

## How It Works

1. **Training Phase**: Analyzes your previous messages to build word frequency + Markov chain
2. **Generation Phase**: Creates replies matching your style
3. **Confidence Scoring**: Rates replies (0-100%) based on how similar to your messages
4. **Auto-Reply**: Only sends replies when confidence ≥ 85% threshold
5. **Logging**: Records all decisions for improvement

## Configuration

Edit `integration_daemon.py`:
- `confidence_threshold` - Only auto-reply above this % (default 0.85 = 85%)
- Add whitelist/blacklist contacts

## Safety

⚠️ **WARNING**: Auto-replying violates WhatsApp ToS. Use at your own risk.

Test locally first. Always keep manual approval enabled for sensitive contacts.
